# ขั้นตอนก่อนตอบคำถาม

## กฎหลัก
**ทุกครั้งก่อนตอบคำถาม ห้ามคิดเองและตอบเอง ต้องเข้าไปดูทักษะในโฟลเดอร์ Skills ก่อนเสมอ**

## ขั้นตอนการทำงาน

### 1. ค้นหา Skill ที่เกี่ยวข้อง
- เข้าไปที่โฟลเดอร์ `antigravity-awesome-skills/skills`
- ค้นหาโฟลเดอร์ Skill ที่เกี่ยวข้องกับคำถามหรืองานที่ต้องทำ
- ใช้ชื่อโฟลเดอร์เพื่อระบุทักษะที่ตรงกับงาน

### 2. อ่านไฟล์ SKILL.md
- เมื่อพบ Skill ที่เกี่ยวข้อง ให้ใช้ `view_file` เพื่ออ่านไฟล์ `SKILL.md` ในโฟลเดอร์นั้น
- ทำความเข้าใจคำแนะนำและแนวทางจาก Skill นั้นๆ

### 3. ปฏิบัติตามคำแนะนำ
- ทำตามขั้นตอนและแนวทางที่ระบุไว้ใน SKILL.md
- ใช้ตัวอย่างและ templates ที่ให้มา (ถ้ามี)

### 4. ตอบคำถามหรือทำงาน
- หลังจากศึกษา Skill แล้ว จึงค่อยตอบคำถามหรือทำงานตามที่ร้องขอ

## ตัวอย่าง Skills ที่ควรตรวจสอบ

| ประเภทงาน | Skills ที่เกี่ยวข้อง |
|-----------|---------------------|

| Testing | `testing-patterns`, `test-driven-development` |
| Architecture | `architecture`, `architecture-patterns`, `senior-architect` |

## หมายเหตุสำคัญ
- ถ้าไม่พบ Skill ที่เกี่ยวข้อง ให้แจ้งผู้ใช้และถามว่าต้องการดำเนินการต่ออย่างไร
- สามารถใช้หลาย Skills ร่วมกันได้ถ้างานต้องการความรู้หลายด้าน
